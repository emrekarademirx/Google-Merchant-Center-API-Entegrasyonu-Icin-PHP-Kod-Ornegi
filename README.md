# Google Merchant Center API Entegrasyonu İçin PHP Kod Örneği

Bu proje, Google Merchant Center'ın API'sini kullanarak ürün verilerini okumak ve düzenlemek için bir PHP örneği içerir.

## Nasıl Kullanılır

1. Google Cloud Console'da bir proje oluşturun ve Merchant Center API'si için kimlik doğrulama anahtarlarınızı oluşturun.

2. `config.php` dosyasında, oluşturduğunuz kimlik doğrulama anahtarlarını ve Merchant Center hesap bilgilerinizi girin.

3. Örnek dosyaları çalıştırmak için PHP'nin yüklü olması gereklidir.

## Örnekler

- `get_products.php`: Merchant Center'da yer alan ürünleri listeler.
- `update_product.php`: Merchant Center'da belirli bir ürünün bilgilerini günceller.

## Daha Fazla Bilgi

Daha fazla bilgi için Google Merchant Center API belgelerine bakabilirsiniz.
